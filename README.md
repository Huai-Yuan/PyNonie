# PyNonie
# An app for me to learn Flask

## usage
```bash
flask --app PyNonie run --debug
flask --app PyNonie init-db  # init db
```

## install
```bash
pip3 install -e .
```

## Build and Install
```bash
pip install build
python -m build --wheel

pip install PyNonie-1.0.0-py3-none-any.whl
```
